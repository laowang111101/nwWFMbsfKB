# Python图书借阅管理系统 python935

## 项目概述

Python图书借阅管理系统是一款用于管理图书借阅流程的软件，具备用户管理、图书信息管理、借阅与归还处理等功能。系统基于稳定的技术栈，确保高效且无故障的运行。

## 技术栈

- **开发语言：** Python
- **后端框架：** Flask
- **前端技术：** HTML、CSS、JavaScript、jQuery
- **数据库：** MySQL
- **开发工具：** PyCharm、Navicat
- **运行环境：** 
  - Python 3.0及以上版本
  - MySQL 8.0

## 功能模块

- **用户模块**
  - 登录注册
- **图书操作**
  - 图书借阅
  - 图书归还
- **信息管理**
  - 图书信息管理
  - 用户管理
- **统计模块**
  - 借阅统计
  - 图书统计

## 系统角色

- 系统管理员：负责用户管理与图书信息管理
- 普通用户：可进行登录注册、图书借阅与归还

## 运行环境

- Python 3.0 或更高版本
- MySQL 8.0
- 支持主流浏览器

## 部署步骤

1. 安装Python环境
2. 安装MySQL数据库
3. 克隆或下载项目代码
4. 安装依赖包
5. 配置数据库连接
6. 运行项目
